# acosh

> ES2015 [`Math.acosh()`](https://developer.mozilla.org/en-US/docs/Web/JavaScript/Reference/Global_Objects/Math/acosh) [ponyfill](https://ponyfill.com)

## Install

```
$ npm install acosh
```


## Usage

```js
const acosh = require('acosh');

acosh(2);
//=> 1.3169578969248166
```
